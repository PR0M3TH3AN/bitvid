# Context: Daily Test Audit

**Goal**: Run a daily audit of the test suite to verify test quality, identify flakiness, and ensure critical paths are covered.

**Scope**:
- Run unit tests with coverage.
- Detect flaky tests by re-running.
- Static analysis for anti-patterns.
- Generate a report.

**Definition of Done**:
- `test-audit-report-YYYY-MM-DD.md` created.
- Logs saved in `test_logs/`.
- Artifacts in `test-audit/`.
