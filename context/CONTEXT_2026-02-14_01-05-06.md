# Context: Test Audit Agent

**Mission:** Run daily audits of the test suite to verify tests actually test the behaviors they claim to, find fragile or missing tests, and propose small, high-confidence fixes.

**Scope:**
- Unit tests (`tests/unit/`, `tests/nostr/`, etc.)
- Coverage analysis
- Flakiness detection
- Static analysis

**Definition of Done:**
- `test-audit/test-audit-report-YYYY-MM-DD.md` created.
- `_completed.md` log created.
- Lint passed.
