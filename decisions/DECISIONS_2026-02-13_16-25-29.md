# Decisions - InnerHTML Migration for js/channelProfile.js

- **Line 2341, 2350, 4561 (`innerHTML = ""`)**: Replace with `textContent = ""` for clearing content. This is safe and faster.

- **Line 4526, 4554, 4910, 5418 (Static "No videos"/"Failed" messages)**:
    - Replace with `textContent` for the text.
    - Create the `<p>` element using `document.createElement`.
    - Apply classes via `className`.

- **Line 5297 (Loading State)**:
    - Create `div` wrapper.
    - Create `span` for text.
    - Append.

- **Line 4134 (Subscribe Button)**:
    - Create `button` element.
    - Create `img` for icon.
    - Create `span` for text (Subscribe/Unsubscribe).
    - Use `textContent` for the dynamic label.
    - Replace the template string with DOM construction.

- **Line 3898 (Zap Controls Template)**:
    - This is a large static block.
    - I will convert this to a series of `createElement` calls appended to a `DocumentFragment` or directly to `panel`.
    - Although it is verbose, it eliminates `innerHTML` entirely and avoids any risk of future injection if variables were added.
    - It ensures strict compliance with the migration goal.
