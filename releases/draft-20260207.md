Both channels have received significant updates introducing new dedicated views for **Kids**, **Watch History**, **Search**, **Subscriptions**, and **Docs**. These additions aim to improve content discovery and user navigation.

Extensive test coverage has been added for Nostr features, UI components, and critical flows (PR #2153), enhancing platform stability.
